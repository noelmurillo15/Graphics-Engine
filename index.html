    3D Graphics Engine Utilizing C++ and DirectX 9

    Project maintained by noelmurillo15 Hosted on GitHub Pages — Theme by mattgraham

    Welcome to My Github Page.

    In this project I worked on a 3D Graphics renderer from scratch. I was able to proceduraly draw an indexed triangle primitive. I also made an objloader that rendered indexed models which were loaded from files using multithreading. I created pixel and vertex shaders for ambient, spot, directional and point lights. I also added normal mapping using the shaders. I was able to draw multiple instances of a tree model and used frustum culling technique to optimize my renderer.
